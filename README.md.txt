# Snip The Bill - Shopify Landing Section
A clean, eco-tech Shopify section that showcases Snip The Bill’s smart energy-saving gadgets.

## How to Install
1. In Shopify Admin, go to Online Store → Edit Code.
2. Add `snip-the-bill-landing.liquid` to your `/sections` folder.
3. Upload `snip-style.css` and your images to `/assets`.
4. Add the section in your Theme Customizer.
